# 数据挖掘导论 8.4 8.5

+ DBSCAN
  + 基于密度的聚类
  + 参数：半径Eps，领域个数阈值MinPts
  + 类型：核心点、边界点、噪声点（被抛弃）
  + 参数选择：观察点到它的k个最近邻距离，找到k-距离急剧下降的距离作为Eps，k为MinPts值
  + 优点：
    + 相对抗噪声，能处理任意形状和大小的簇
  + 缺点：
    + 簇的密度变化过大时，参数不好确定
    + 高维数据的密度定义（下一章）
    + 计算量大
    
+ 簇评估
  + 基于密度的不一定是球形的簇，所以SSE（考虑点距离）不行
  + 评估方法：
    + 非监督（内部指标）：凝聚性、分离性
    + 监督（外部指标）：熵、jaccard、纯度
    + 相对
